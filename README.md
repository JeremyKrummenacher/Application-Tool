## Getting Started

### Run Application

run:
  ```sh
  clone repository
  ```
   ```sh
  cd /admin
  ```
  ```sh
  npm install
  ```
  ```sh
  npm run dev
  ```

### Prequisites

npm:

  ```sh
  npm install npm@latest -g
  ```

### Installation

Vue.js:

 ```sh
  npm init vue@latest
  ```
  
Vite

 ```sh
  npm create vite@latest
  ```
  
Pinia:

```sh
  npm install pinia
  ```
  
Vue Router:

```sh
  npm install vue-router@4
  ```

Quasar Framework:

```sh
  npm install -g @quasar/cli
  ```
  
TypeScript:

```sh
  npm install typescript --save-dev
  ```
  
Docker Compose:

  [Installation Instructions](https://docs.docker.com/compose/install/)
